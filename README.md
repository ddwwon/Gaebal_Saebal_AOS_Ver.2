## 💡 Project Information
개발 공부를 기록할 수 있는 다이어리 앱<br/><br/>
1️⃣ 간단한 기록 작성(글, 사진)  

2️⃣ 개발자 맞춤형 기록 작성(Github 연동, Baekjoon 연동, Code 작성)  
###### &nbsp;&nbsp;&nbsp;&nbsp;※ 인터넷(WiFi, 데이터) 연결 필요  
- Github Access  Token을 사용해 사용자의 Private Repository의 Pull Request, Issue, Commit을 불러올 수 있습니다.  
- Baekjoon 문제 번호를 입력하면 해당 문제 번호와 문제 이름을 불러올 수 있습니다.  
- 짧고 간단한 Code를 작성할 수 있습니다.  

3️⃣ 폴더로 기록 관리  
- 언어, 플랫폼 등 사용자가 원하는 분야로 폴더를 분류하여 기록을 관리할 수 있습니다.  

4️⃣ Tag를 활용한 검색  
- 각 기록의 Tag를 등록해 검색 화면에서 제목, Tag의 내용으로 검색이 가능합니다.
